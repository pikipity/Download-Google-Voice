Introduction

This program is used to download the google voice in the "Google Translation".

It can be used in Windows, Linux and Mac. For Mac and Linux, you can run this program by running the command "python main.py" in the direction which included "main.py". For Windows, there will be the "setup.exe" program to allow you set up this program easily.

But in Linux, I didn't test it. Maybe there will be some problems. If you find any problems, please feel free to contect me.

Now, It only support "Chinese", "English" and "Japanese". For "Japanese", I didn't test in Windows. Maybe there will be some problems. If you find some problems, please feel free to contect me.

Have Fun. Thank you.

Installation

You can download this project from there. For Windows user, you can use setup.exe to install this program or directly run main.py if you have install Python language in your computer. For Linux and OSX users, you can direactly run main.py to start the program by using the follow command:

python path_of_your_installation/main.py
